FILE DESCRIPTIONS:
==================

bot.py
------
The main Discord bot application. Contains:
- Discord.py bot setup
- PosterGenerator class (creates images)
- Slash command handlers
- Image processing logic
Run with: python3 bot.py

requirements.txt
----------------
Python packages needed:
- discord.py (Discord API)
- Pillow (Image generation)
- aiohttp (Async HTTP)
- python-dotenv (Environment vars)
Install with: pip install -r requirements.txt

.env.example
------------
Template for your bot token.
Usage:
1. Copy to .env: cp .env.example .env
2. Edit .env with your actual token
3. Keep .env private (never commit to git)
